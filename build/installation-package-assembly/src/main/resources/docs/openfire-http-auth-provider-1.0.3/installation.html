Installation

  1. Install Openfire with your preferred fallback authentication provider. If you don't need a fallback use the internal mechanism for installation. You have to install Openfire before you can configure the HTTPRequestAuthProvider.
  2. Configure the HTTPRequestAuthProvider
  3. Switch the provider manually
    1. Stop Openfire,
    2. Connect to the database, which is used by Openfire manually,
      1. Go to table ofproperty,
        1. Change value of provider.auth.className to de.communardo.xmpp.openfire.auth.HTTPRequestAuthProvider,
        2. Change value of provider.user.className to org.jivesoftware.openfire.user.DefaultUserProvider,
    3. Start Openfire. Now the HTTPRequestAuthProvider should be used.

    Hint: An alternative way to change these settings would be via the Openfire administration page. Go to http://yourOpenfire:9090/server-properties.jsp and change the settings there.
